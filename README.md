# Toronto Downtown Runner 🏃‍♂️🏙️

A high-speed, 3D infinite runner game set in the vibrant streets of downtown Toronto. Dodge traffic, collect coins, and see how far you can run through the 6ix!

## 🎮 How to Play

### Controls
- **Desktop:**
  - `Left Arrow` or `A`: Move Left
  - `Right Arrow` or `D`: Move Right
  - `Space`: Jump (Coming Soon)
- **Mobile:**
  - `Swipe Left`: Move Left
  - `Swipe Right`: Move Right

### Objective
- Run as far as possible to increase your score.
- Collect **Gold Coins** to boost your score.
- Avoid **Cars** and obstacles. Hitting one ends the game!
- The game gets faster the longer you survive.

## ✨ Features
- **Toronto Themed:** Experience a stylized version of downtown Toronto with iconic city vibes.
- **Dynamic Speed:** The game difficulty scales as your speed increases over time.
- **Responsive Design:** Fully playable on both desktop and mobile devices.
- **3D Graphics:** Powered by Three.js for a smooth, immersive experience.

## 🛠️ Tech Stack
- **Framework:** [React](https://reactjs.org/)
- **3D Engine:** [React Three Fiber](https://docs.pmnd.rs/react-three-fiber) & [Three.js](https://threejs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) (for UI)
- **Icons:** [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🖼️ Customization
To use your own character avatar:
1. Upload your images to the `/src` directory.
2. Update the `textures` paths in `src/Game/Player.tsx`.
